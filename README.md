# Data Cleaning & Reporting Automation

Cleaned and analyzed a modified Titanic dataset using Python.

## What I did
- Found 7 duplicate rows and removed them
- Filled missing values in Age (used median), Cabin and Embarked columns
- Fixed inconsistent text like female/Female and q/Q in Embarked
- Generated summary stats and 4 charts automatically

## Files
- `clean_data.py` - cleans the raw dataset
- `report.py` - generates charts and report
- `cleaned_data.csv` - dataset after cleaning
- `summary_report.csv` - stats report

## Libraries
pandas, matplotlib

Titanic dataset - 891 rows, 12 columns# DATA_CLEANING_AUTOMATION
